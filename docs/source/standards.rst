Standards
==========

.. _ogc-pixel-is-area:

Pixel is Area
-------------

Convention defining pixel anchoring at the cell upper left corner.

Official documentation : https://docs.ogc.org/is/19-008r4/19-008r4.html#_pixelisarea_raster_space


.. _ogc-pixel-is-point:

Pixel is Point
--------------

Convention defining pixel anchoring at the cell center rather than the corner.

Official documentation : https://docs.ogc.org/is/19-008r4/19-008r4.html#_pixelispoint_raster_space
